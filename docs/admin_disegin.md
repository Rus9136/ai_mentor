Комплексные рекомендации по дизайну и структуре:

## 🎨 **1. Общая Концепция Дизайна**

### Принципы дизайна для образовательной платформы:
- **Чистота и минимализм** - избегайте визуального шума
- **Информационная иерархия** - важное должно быть заметным
- **Контекстная навигация** - пользователь всегда знает где находится
- **Быстрый доступ к действиям** - минимум кликов до цели

- **Бейджи** для уведомлений (например: "3 новых школы")

## 🌲 **7. Редактор структуры учебника**

### **Древовидная структура с drag-and-drop:**

```
┌─────────────────────────────────────────────────────┐
│ Структура учебника: Алгебра 7 класс                 │
│ [+ Добавить главу]    [💾 Сохранить]  [👁️ Превью]   │
├─────────────────────────────────────────────────────┤
│                                                     │
│  📘 Алгебра 7 класс                         [▼]   │
│    ├─ 📂 Глава 1: Рациональные числа       [⚙️ ✕]  │
│    │   ├─ 📄 1.1 Положительные числа       [⚙️ ✕]  │
│    │   │      стр. 12-15 | ⭐⭐⭐☆☆                │
│    │   ├─ 📄 1.2 Модуль числа              [⚙️ ✕]  │
│    │   │      стр. 16-20 | ⭐⭐⭐⭐☆               │
│    │   └─ [+ Добавить параграф]                   │
│    │                                               │
│    ├─ 📂 Глава 2: Выражения               [⚙️ ✕]  │
│    │   └─ [+ Добавить параграф]                   │
│    │                                               │
│    └─ [+ Добавить главу]                          │
│                                                     │

```

### **Функциональность:**
- **Drag & Drop** для изменения порядка
- **Раскрывающееся дерево** (accordion)
- **Inline редактирование** названий
- **Быстрые действия** (⚙️ редактировать, ✕ удалить)
- **Визуальная индикация** уровней (отступы, линии)
- **Цветовое кодирование** (глава, параграф, тест)

## 📝 **8. Редактор контента (Rich Text Editor)**

### **Рекомендуемые библиотеки:**
- **TipTap** (современный, расширяемый)
- **Quill** (легкий, стабильный)
- **ProseMirror** (мощный, гибкий)

### **Необходимые функции:**
```
┌─────────────────────────────────────────────────────┐
│ [B][I][U] [H1][H2][H3] [•][1.] [🔗][📷][📊][∑ƒx]    │
├─────────────────────────────────────────────────────┤
│                                                     │
│  Модуль числа - это...                            │
│                                                     │
│  Формула: ∫ |x| = x² + C                          │
│                                                     │
│  [Вставленное изображение]                         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### **Особенности:**
- **LaTeX поддержка** для математических формул
- **Вставка изображений** с превью и размером
- **Таблицы** с редактированием
- **Синтаксис highlight** для кода
- **Автосохранение** каждые 30 секунд
- **История изменений** (undo/redo)

## ⚡ **12. UX Best Practices**

### **Быстрый доступ:**
- **Поиск** глобальный (Cmd+K / Ctrl+K)
- **Недавние действия** в header
- **Избранное** (закрепленные разделы)
- **Хлебные крошки** для навигации

### **Обратная связь:**
- **Loading состояния** (скелетоны)
- **Уведомления** о действиях
- **Прогресс бары** для загрузок
- **Подтверждения** критических действий

### **Помощь пользователю:**
- **Tooltips** для кнопок
- **Пустые состояния** (empty states) с инструкциями
- **Онбординг** для новых пользователей
- **Справка** (? иконка) в сложных разделах
